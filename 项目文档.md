# 校里·约拍 - 前端项目文档

## 一、项目概述

**项目名称：** 校里·约拍  
**项目类型：** 微信小程序（基于 UniApp + Vue3）  
**核心功能：** 校园摄影师展示与约拍服务平台  
**目标用户：** 大学生（找摄影师）、校园摄影师（接单）

**核心价值：**
- 为大学生提供便捷的校园摄影师发现与筛选服务
- 为校园摄影师提供作品展示与接单管理平台
- 打造校园摄影生态圈

---

## 二、技术栈

### 前端技术
- **框架：** UniApp + Vue3（Composition API）
- **开发语言：** JavaScript + SCSS
- **状态管理：** Vue3 Reactive（轻量级全局状态）
- **UI设计：** 自定义组件 + 自适应布局

### 开发环境
- **编译器版本：** 4.87（vue3）
- **构建工具：** Vite v5.2.8
- **开发端口：** 默认 5173（自动递增）

## 四、页面功能详解

### 4.1 首页

**文件路径：** `src/pages/index/index.vue`

**功能描述：**
- 摄影师列表展示（新入驻 / 接单中）
- 筛选功能（擅长风格 + 价格区间）
- 收藏功能
- 卡片点击跳转详情

**核心数据：**
```javascript
// 摄影师卡片数据结构
{
  id: '1',                    // 摄影师ID
  group: 'active',            // 分组：'new'新入驻 / 'active'接单中
  name: '开到茶花',           // 昵称
  avatar: 'url',              // 头像URL
  subtitle: '热爱胶片摄影',   // 副标题
  price: '300-800',           // 价格区间
  bio: '个人简介...',         // 简介
  styles: ['日系', '胶片'],   // 擅长风格标签
  statusLabel: '接单中',      // 状态标签
  shots: ['url1', 'url2'],    // 作品展示图片
  liked: true,                // 是否收藏
  wechatId: 'shiro_001'       // 微信号
}
```

**筛选功能：**
- 擅长风格：11个标签（人像摄影、胶片复古、日系清新、毕业写真、情绪片、互勉、证件照、棚拍、日系、女摄、同校）
- 价格区间：默认 0-588，最高可设置到 9999
- 筛选徽章：显示当前激活的筛选条件数量

**交互逻辑：**
1. 切换标签页（新入驻 / 接单中）
2. 点击筛选按钮 → 弹出筛选面板
3. 选择擅长风格 + 拖动价格滑块 → 点击"完成"应用筛选
4. 点击卡片 → 跳转到摄影师详情页
5. 点击收藏按钮 → 添加/移除收藏

## 六、状态管理

**文件路径：** `src/store/app.js`

### 6.1 全局状态（appState）

```javascript
export const appState = reactive({
  likedIds: ['1'],              // 收藏的摄影师ID列表
  auditStatusValue: 'passed',   // 入驻审核状态
  meAvatar: 'https://...',      // 我的头像
  meName: '食盐 shiro',         // 我的昵称
  orderEnabled: true,           // 接单状态开关（摄影师身份）
  currentDetailCard: null,      // 当前查看的详情卡片
  isLoggedIn: false             // 登录状态
})
```

### 6.2 状态管理函数

```javascript
// 判断是否已收藏
export function isLiked(id) {
  return appState.likedIds.includes(String(id))
}

// 切换收藏状态
export function toggleLike(id) {
  const s = String(id)
  const i = appState.likedIds.indexOf(s)
  if (i >= 0) {
    appState.likedIds.splice(i, 1)
  } else {
    appState.likedIds.push(s)
  }
}

// 设置审核状态
export function setAuditStatus(value) {
  appState.auditStatusValue = value
}

// 设置个人资料
export function setMeProfile({ avatar, name }) {
  if (avatar) appState.meAvatar = avatar
  if (name !== undefined) appState.meName = name
}

// 设置登录状态
export function setLoggedIn() {
  appState.isLoggedIn = true
}
```

## 十、总结

**校里·约拍**是一个专注于校园摄影服务的微信小程序，采用 UniApp + Vue3 技术栈开发，具有以下特点：

✅ **双身份模式**：用户可以切换"找约拍"和"摄影师"两种身份

✅ **智能筛选功能**：擅长风格 + 价格区间组合筛选  
✅ **收藏管理系统**：一键收藏、收藏列表查看  
✅ **登录态管理**：基于微信手机号一键登录  
✅ **协议与规则**：完善的用户协议、隐私政策、入驻协议
